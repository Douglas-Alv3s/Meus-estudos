## Mudando o fluxo de um laço

>> Para interromper o fluxo do laço utilizase o comando `continue;`
>> usando esse comando o laço é interrompido e voltado para a base
>> novamente

>> Para interromper o fluxo do laço utilizase o comando `break;;`
>> usando esse comando o laço é interrompido e sai do laço


## REPETIÇÃO COM TESTE NO FINAL
```java
int cc = 0;
do{
    System.out.println("Cambalhota");
    cc++;
} while(cc>=4);

```
